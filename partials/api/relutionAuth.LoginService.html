relutionAuth:LoginService

Description

Simple Service to login and logout on relution server and store the result in the LoginService.userResponse

Dependencies

  • $q

    , $state , Base64, $rootScope, $relutionAuthLauncher

Methods

  • clearCredentials()

    remove the Basic auth from the Header

  • error()

    login failed

  • errorLogout()

    logout failed

  • logon()

    set credentials the use post to connect

  • secureLogin()

    login on Server with Credentials

  • secureLogout()

    logout on Server

  • setPassword()

    set the password

  • setUsername()

    set the username

  • success()

    login succesfully

  • successLogout()

    logout succesfully

Properties

  • basicAuth

    a base64 hash

  • form

    the form fields

  • isLoggedIn

    bool if the user is loggedIn

  • userResponse

    the response user
